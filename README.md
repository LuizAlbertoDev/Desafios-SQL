# 🗄️ Desafios SQL

Repositório de desafios práticos para aprendizado de SQL com SQLite e Node.js, desenvolvidos como parte de uma jornada de transição de carreira para desenvolvimento fullstack.

## 🎯 Objetivo

Aprender SQL do zero através de desafios progressivos — cobrindo os principais comandos e conceitos exigidos no mercado de trabalho.

## 📚 Conteúdo

| Desafio | Conteúdo |
|---------|----------|
| 001 | Introdução ao SQLite — CREATE TABLE, INSERT, SELECT |
| 002 | Filtros com WHERE |
| 003 | UPDATE e DELETE |
| 004 | Reset do banco — DROP TABLE |
| 005 | ORDER BY e LIMIT |
| 006 | CRUD completo — projeto de produtos |
| 007 | Busca com LIKE |
| 008 | COUNT, SUM e GROUP BY |
| 009 | JOIN com chave estrangeira |
| 010 | Projeto final — sistema de loja com clientes e pedidos |
